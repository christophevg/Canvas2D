Rounded Corners

This exmaple shows how to add rounded corners to a standard rectangle.

Rounded Corners

      +dynamic sheet mySheet {
        [@100,75]
        rectangle myRounded +geo="100x100" +roundCorners 
          +fillColor="rgb(255,240,175)" +lineColor="rgb(250,125,0)"
          +label="Hello Roundings" +labelPos="top-inner" 
          +labelColor="rgb(250,125,0)";
      }
