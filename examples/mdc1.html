MDC 1 : Basic Usage

The example is taken from the excellent Canvas Tutorial by the Mozilla developers and implemented both using Javascript as the Canvas2D DSL.
See: https://developer.mozilla.org/en/Canvas_tutorial/Basic_usage.

HTML5 Canvas API

var manager = new Canvas2D.Manager();
var canvas  = manager.setupBook("myCanvas1");
var ctx     = canvas.addSheet();
manager.startAll();

ctx.fillStyle = "rgb(200,0,0)";
ctx.fillRect (10, 10, 55, 50);

ctx.fillStyle = "rgba(0, 0, 200, 0.5)";
ctx.fillRect (30, 30, 55, 50);

Canvas2D API

sheet mySheet {
  [@10,10] rectangle rect1 +geo="55x50" +fillColor="rgb(200,0,0)";
  [@30,30] rectangle rect2 +geo="55x50" +fillColor="rgba(0,0,200,0.5)";
}
