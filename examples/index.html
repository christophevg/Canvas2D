Canvas2D Examples

  • styles introduces the different ways Canvas2D can be incorporated in a webpage.
  • lines introduces the different ways Canvas2D can draw lines.
  • custom connectors shows that the connector shape can be customized.
  • DSL shows how the ADL language can be used to define the shapes on the diagram.
  • Short DSL shows the shorthand version of the same ADL code.
  • Kickstart does everything for you: detecting canvas elements and turning them into a Canvas2D
  • Text shows the text rendering possibilities.
  • Images shows how to introduce an image on the Canvas as a Shape
  • Aliasses allow definition of own shapes, completely through ADL.
  • Crisp Lines Support showing how try to Do Whay You Mean when using Canvas2D
  • Arrow shape shows how to use a custom shape
  • iPhone application-style HTML page to show off we now also support it ;-)
  • composite shapes allow the composition of complex shapes with little effort.
  • custom routing shows that the routing of connectors can be customized.
  • new connectors shows the new implementation of the old connectors using the new routing algo.
  • label connectors shows how to add labels to connectors.
  • round corners are a new property of Rectangles.

Canvas2D Integration Examples

  • multiple sheets offer access to multiple virtual canvasses
  • source and console shows that you can bind your source, the generated source and the console to your own HTML elements.
  • libraries & shapes shows how to interrogate Canvas2D for known libraries and shapes.
  • errors shows how errors are handled

Using Canvas2D to do More

  • grapher shows how to use Canvas2D to create small graphs on the fly with 5 lines of Javascript code.
