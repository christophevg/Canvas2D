Canvas2D Examples

  • styles introduces the different ways Canvas2D can be incorporated in a webpage.
  • lines introduces the different ways Canvas2D can draw lines.
  • custom connectors shows that the connector shape can be customized.
  • DSL shows how the ADL language can be used to define the shapes on the diagram.
  • Short DSL shows the shorthand version of the same ADL code.
  • Kickstart does everything for you: detecting canvas elements and turning them into a Canvas2D
  • Text shows the text rendering possibilities. See also our page on the performance issues when rendering text on IE
  • Images shows how to introduce an image on the Canvas as a Shape
  • Aliasses allow definition of own shapes, completely through ADL.
